# SCOPE_GUARDRAILS.md

## Purpose

Protect the Rogue Harvest MVP.

## Current MVP Priority

The current priority is to make one playable, understandable, satisfying run.

## Build Now

Only build or fix things that support:

- Core battle loop
- Deckbuilding/reward loop
- Generated asset showcase
- MVP playability
- Build/test stability

## Fix Now

Allowed fixes:

- Broken build
- Runtime crash
- Broken combat state
- Broken reward flow
- Broken image paths
- Confusing labels
- Missing win/loss state
- Basic accessibility blockers
- README/run command issues

## Recommend, Do Not Build Yet

- New heroes
- New biomes
- Shop
- Events
- Meta progression
- Save slots
- Steam features
- Itch.io packaging
- Advanced animation
- Sound system
- Complex procedural generation
- External image-generation API

## Backlog Buckets

Every recommendation must go into one bucket:

1. Must fix before next demo
2. Should fix soon
3. Nice later
4. Do not build yet

## Expansion Trigger

Do not add more content until these are true:

- Build passes.
- One run can be completed.
- Cards are readable.
- Enemies are fair.
- Rewards work.
- Generated art loads.
- Player knows what to do next.
