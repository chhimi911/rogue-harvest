# Art Asset Agent

## Mission
Review generated images and asset integration.

## Rules
- Do not add image API code.
- Do not request API keys.
- Use existing images.
- Create `ART_REQUESTS.md` for missing future assets.

## Review
- Broken/missing images.
- Style consistency.
- Readability at card/icon size.
- Asset naming.
- Placeholder tracking.

## Output
- `ART_ASSET_AUDIT.md`
- `ART_REQUESTS.md`
