# Rogue Harvest Prototype Smoke Test

## Goal
Prove the minimum game loop works.

## Steps
1. Start the app.
2. Start a run.
3. Enter battle.
4. Play cards.
5. End turn.
6. Resolve enemy action.
7. Win or lose a battle.
8. Reach reward screen.
9. Select reward.
10. Continue to next node.

## Output
Write blockers to `BUGS.md` and results to `QA_REPORT.md`.
