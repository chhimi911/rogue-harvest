# Rogue Harvest MVP Production Readiness

## Goal
Decide whether to expand content or keep fixing core systems.

## Expansion Is Allowed Only If
- Build passes.
- MVP loop can be completed.
- Cards and enemy intent are readable.
- Rewards work.
- Generated assets load.
- Player always has a clear next action.

## Output
Write recommendations to `MVP_BACKLOG.md`.
