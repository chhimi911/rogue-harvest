# Rogue Harvest Launch Readiness Check

## Goal
Prepare a future launch checklist without doing launch work yet.

## Check
- App builds.
- Public demo loop works.
- Critical bugs are gone.
- Art is acceptable for a public prototype.
- README explains local run/build.
- Known limitations are documented.

## Output
Write future launch requirements to `MVP_BACKLOG.md`.
