# Rogue Harvest Concept Lock Review

## Goal
Confirm that the current build still communicates Rogue Harvest clearly.

## Check
- Title and presentation say Rogue Harvest.
- Fantasy is anthropomorphic produce vs the Blight.
- Starter hero is Blackberry Rogue or clearly equivalent.
- Enemies feel like garden corruption, pests, rot, mold, or pantry mutations.
- Cards feel botanical, rogue-like, poison/thorn/dodge flavored.
- Run has a clear win/lose condition.

## Output
Write a `QA_REPORT.md` section called `Concept Lock`.
