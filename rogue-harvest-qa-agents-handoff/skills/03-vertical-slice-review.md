# Rogue Harvest Vertical Slice Review

## Goal
Judge if the current build feels coherent and demo-worthy.

## Check
- One complete Starter Patch run exists.
- Hero, enemies, cards, relics, and background feel related.
- Generated art loads.
- Battle and reward screens are visually consistent.
- Boss encounter feels distinct.
- UX explains itself enough for a first-time player.

## Output
Write findings to `QA_REPORT.md` and `RECOMMENDATIONS.md`.
